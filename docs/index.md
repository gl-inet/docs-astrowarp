# What is AstroWarp 

AstroWarp is an advanced networking platform designed to seamlessly connect multiple GL.iNet routers into a unified network, enabling effortless remote access. With flexible networking capabilities, user-friendly configuration and robust privacy protection, AstroWarp delivers secure and reliable connectivity at all times. Whether for remote work or secure access to internal corporate resources across multiple sites, AstroWarp is your ideal solution for high-performance, protected networking.
